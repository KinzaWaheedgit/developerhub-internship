# Developerhub Internship - E-Commerce Frontend Design

This is my assignment code for the internship, built using 
HTML, CSS, and JavaScript. 
The file structure and organization are detailed below:

## 📂 Project Structure & Task Details
* Task 1: `index.html` (Home Page)
* Task 2:`listing.html` (Product Listing Page)
* Task 3: `product-details.html` (Product Details Page)

* **`assets/` Folder:** Contains all the icons and images used across the pages, which have been exported directly from Figma.
